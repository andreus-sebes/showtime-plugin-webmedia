webMedia plugin for Showtime - XML file builder

Just start with "Sources" tab and then go to next tab and so on.

Please note that this tool only creates a xml file. For editing your file you can also use this tool to make the xml "<item>" tags and then manualy add then to your existing xml file.

  • Sources
  • Publishers
  • Themes
  • Countries
  • Types
  • Create XML

1. Add your sources

Source URL examples:

- RSS feed: http://rss.cnn.com/rss/edition.rss

- Imagecast: http://picasaweb.google.com/data/feed/base/all?alt=rss

- Podcast: http://rss.cnn.com/services/podcasting/newscast/rss.xml

- Videocast: http://rss.cnn.com/services/podcasting/ac360/rss.xml

- Live TV: use .strm URL or see Tutorial: Get RTMP links for TV Streams and others (Showtime) for rtmp links

- Live Radio: use .strm URL or see Tutorial: Get RTMP links for TV Streams and others (Showtime) for rtmp links

[+] [-]

2. If you want images in the publishers (automatically populated based on your sources), for each one put an image URL

3. If you want images in the themes (automatically populated based on your sources), for each one put an image URL

4. Countries are automatic populated by default, don't need to change

5. Types are automatic populated by default, don't need to change

6. Create your XML file and submit it to Pastebin OR Pastehtml

Pastebin

Form for creating a xml file

 

 

After submit, in pastebin, download your XML file OR copy the raw xml URL to your webMedia settings in Showtime

Pastehtml

Form for creating a xml file

 

After submit, in pastehtml, save your XML file OR copy the resulting URL to your webMedia settings in Showtime
